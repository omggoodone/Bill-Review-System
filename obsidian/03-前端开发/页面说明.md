# 页面说明

## 登录与认证

### 登录页 (`views/login.vue`)
- 用户名/密码/验证码三要素
- "记住我"功能：密码通过 jsencrypt RSA 加密存 Cookie
- 成功 → 存 token → 获取用户信息 → 生成动态路由 → 按角色跳转

### 注册页 (`views/register.vue`)
- **审核制注册**: 邮箱 + 申请说明 + 选择角色（普通用户/审核员）
- POST `/api/register-request` → 提交申请进入待审 → 提示等待管理员审批
- 管理员审批通过后生成初始账号密码，用户凭此登录

## 首页仪表盘

首页已废弃 `/index` 路由。登录后按角色跳转:
- 系统管理员 → `/admin/dashboard`（管理概览）
- 审核员 → `/bill/review`（待审核列表）
- 普通用户 → `/bill/manage`（票据概览）

## 业务模块

### 票据管理 (`views/biz/bill/`)

**票据概览 (`manage/index.vue`)** — 根据角色展示不同内容：
- **普通用户**: 个人统计卡片（总数/草稿/待审/已通过/已退回）+ ECharts 分类饼图 + 类别金额柱状图 + 快捷操作
- **审核员**: 审核统计卡片（待审核/今日通过/今日退回/通过率/积压预警>3天）+ 最近审核记录表

**我的票据 (`index.vue`)** — 普通用户票据列表：
- 状态 Tab 切换（全部/草稿/待审核/已通过/已退回）
- 创建/编辑/提交/删除操作
- 状态颜色: 草稿(灰)、已提交(蓝)、已通过(绿)、已退回(红)

**票据审核 (`review/index.vue`)** — 审核员审核队列：
- 待审核票据列表，支持按提交人(模糊)、类别、金额范围筛选
- 通过/退回操作，积压>3天黄色高亮

**组件 (`components/BillForm.vue`)** — 票据创建/编辑弹窗：
- 标题、金额、类别下拉、描述、附件上传

### 管理员页面 (`views/biz/admin/`)

| 页面 | 组件 | 说明 |

## 全局指令

| 指令 | 用法 | 说明 |
|------|------|------|
| `v-hasPermi` | `v-hasPermi="['system:user:add']"` | 无权限时移除 DOM 元素，`*:*:*` 超级管理员绕过 |
| `v-hasRole` | `v-hasRole="['admin']"` | 无角色时移除 DOM 元素，`admin` 角色绕过 |
| `v-copyText` | `v-copyText="text"` | 点击复制文本到剪贴板 |
